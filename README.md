## 积分排名计算方法

### 积分计算方法

> 对于每一场比赛，我们都会计算一次积分，积分计算方法如下：

$$
\text{score} = \text{max}(100, 比赛期间得分 + 比赛后补题分)
$$

#### 比赛期间得分

> 此排名以在对于平台 _比赛结束时_ 的排名为准

|  排名区间  | 得分  |
| :--------: | :---: |
|  $(0,20]$  | $100$ |
| $(20,40]$  | $90$  |
| $(40,60]$  | $80$  |
| $(60,80]$  | $70$  |
| $(80,100]$ | $60$  |

#### 比赛后补题分

> 比赛后补题分截止于此次比赛的一周后的 18:00

$$
比赛后补题分 = 补题数\times 6
$$
